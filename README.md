# k100Sim
Geant4 simulation for the K100 cryogenic setup at UMN

## Requirements
Geant4 10.06\
ROOT 6.22/6.24\
cmake greater than 3.8 less than 3.21

## Instructions
`git clone -b k100sim_BoronShield git@github.com:spandeyehep/k100Sim-1.git ./k100Sim`\
`mkdir build`\
`cd build`\
`cmake ../k100Sim`\
`./k100Sim -r <macro>`
